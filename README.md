# MotorPH Payroll System

## Description
This project is a Java-based payroll system developed for the MO-IT101 course.  
It calculates employee payroll using time-in and time-out records from an Excel file.

## Features
- User login authentication
- Reads employee data from Excel
- Calculates employee working hours
- Computes payroll based on recorded hours

## Technologies Used
- Java
- Apache POI (for Excel file reading)

## Files
PayrollSystemExcel.java – Main program  
MotorPH_Employee_Data.xlsx – Employee data source
